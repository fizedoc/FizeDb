odbc()
------
ODBC方式构造

.. code-block:: php

  public static function odbc (
      string $user,
      string $pwd,
      string $sid,
      mixed $port = "",
      string $charset = "utf8",
      string $driver = null
  ) : \fize\db\realization\oracle\mode\Odbc


:参数:
  +--------+-----------------------------------------+
  |名称    |说明                                     |
  +========+=========================================+
  |user    |用户名                                   |
  +--------+-----------------------------------------+
  |pwd     |用户密码                                 |
  +--------+-----------------------------------------+
  |sid     |连接串                                   |
  +--------+-----------------------------------------+
  |port    |端口号，选填，Oracle默认是1521           |
  +--------+-----------------------------------------+
  |charset |指定编码，选填，默认utf8                 |
  +--------+-----------------------------------------+
  |driver  |指定ODBC驱动名称。                       |
  +--------+-----------------------------------------+
