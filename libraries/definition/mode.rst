======
模式
======


::

    模式需要实现 getInstance() 方法来进行模式的统一调用


+-------------+---------------------+
|属性         |值                   |
+=============+=====================+
|命名空间     |fize\\db\\definition |
+-------------+---------------------+
|类名         |Mode                 |
+-------------+---------------------+


:方法:


+-----------------+----------------+
|方法名           |说明            |
+=================+================+
|`getInstance()`_ |数据库实例      |
+-----------------+----------------+


方法
======
getInstance()
-------------
数据库实例

.. code-block:: php

  abstract public static function getInstance (
      string $mode,
      array $config
  ) : \fize\db\definition\Db


:参数:
  +-------+----------------------+
  |名称   |说明                  |
  +=======+======================+
  |mode   |连接模式              |
  +-------+----------------------+
  |config |数据库参数选项        |
  +-------+----------------------+
